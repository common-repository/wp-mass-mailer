== Description ==

This plugin can send mail to all your WordPress commenter, registered user and the specified post's comments.

[Plugin Homepage](http://linquan.name/1156/ "Plugin Homepage")

== Installation ==

**Upgrade**

1. De-activate the plugin.
2. Delete the old version `wp-mass-mailer` directory

**Install**

1. Unzip the `wp-mass-mailer.zip` file.
2. Upload `wp-mass-mailer` folder to the `/wp-content/plugins/` directory
